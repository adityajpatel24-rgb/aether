# Aether LLC Website

A modern website for Aether LLC. This company offers custom software solutions and sells software products.

## Structure

- `index.html` — Home/landing page
- `about.html` — About the company
- `services.html` — Custom software services
- `shop.html` — Software products for sale
- `contact.html` — Contact & request form
- `styles/` — CSS files
- `scripts/` — JavaScript if needed
